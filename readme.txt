В папке Tet6 находится мобильное приложение, запускается командой npm run start. Возможно, понадобится доустановить пару пакетов для
запуска.
Если при авторизации написать admin@gmail.com 1234, то появится ссылка на админ-панель. В полях Блок вписываете 1 или 2, это выбирает, в какой инфо-блок будут отправлены данные, заполняете остальные поля данными, нажимаете кнопку и вуаля.
В папке server есть сервер, запускается через node registration.js. В начале файла замените имя пользователя и еще пару данных на свои для бд.
В папке web веб-версия мобильного приложения, функционал тот же (нет смены темы), но язык меняется вручную, а не в зависимости от системы, запускается через
npm start
И самое главное - в Tet6 src/app/ip/config.js и в web src/ip/index.js, в них обоих есть const IP_ADDRESS = "192.168.100.3";
Пожалуйста, введите туда свой ipV4, сервер никуда не залит. Если все работает, а через время появляется ошибка и не отображаются новости - значит, сменился ipV4 вашей сети, у меня с сетью такое бывает, не знаю, нормально это или нет.
